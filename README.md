# Python-HTTP-Server

这是一个用 Python 编写的 HTTP 文件服务器，让你可以通过浏览器访问电脑上的文件。基于 Python 标准库 `http.server` 开发，**零第三方依赖**，开箱即用。

## ✨ 功能特性

| 功能 | 说明 | 状态 |
|------|------|------|
| 📁 **静态文件服务** | 提供 HTTP 访问本地文件 | ✅ |
| 🌐 **局域网共享** | 同网络设备均可访问 | ✅ |
| 🔍 **自定义 404** | 美观的错误页面，非默认文本 | ✅ |
| 📝 **彩色日志** | 控制台彩色输出，状态码分色 | ✅ |
| 📄 **文件日志** | 可选写入 log 目录 | ✅ |
| ⚙️ **外部配置** | conf.txt 管理所有设置 | ✅ |
| 🚀 **自动初始化** | 自动创建所需目录和文件 | ✅ |
| 🎨 **跨平台** | Windows / Linux / macOS | ✅ |
| 📦 **打包支持** | 一键打包为 exe 可执行文件 | ✅ |
| 🔒 **CORS 支持** | 默认允许跨域访问 | ✅ |

## 🚀 快速开始

### 环境要求

- **Python 3.6+**（Windows 建议 3.8+）
- 无需安装任何第三方库

### 启动方式

```bash
# 方式 1：直接运行（推荐端口 8080）
python server.py

# 方式 2：使用 80 端口（需要管理员权限）
# Windows（管理员 PowerShell）:
python server.py

# Linux / macOS:
sudo python3 server.py
```

## ⚙️ 配置文件详解
配置文件 conf.txt 使用 INI 格式，首次运行自动生成。

| 配置项                 | 类型  | 默认值                  | 说明     |
| ------------------- | --- | -------------------- | ------ |
| `port`              | 整数  | 8080                 | 服务端口   |
| `enable_access_log` | 布尔  | true                 | 控制台日志  |
| `log_to_file`       | 布尔  | false                | 文件日志开关 |
| `log_file`          | 字符串 | server.log           | 日志文件名  |
| `directory_listing` | 布尔  | true                 | 目录浏览开关 |
| `cors_origin`       | 字符串 | \*                   | 跨域设置   |
| `bind_address`      | 字符串 | 0.0.0.0              | 绑定 IP  |
| `index_files`       | 列表  | index.html,index.htm | 首页候选   |

## 日志级别说明

| 级别      | 颜色    | 含义      |
| ------- | ----- | ------- |
| `INFO`  | 🔵 蓝色 | 一般信息    |
| `OK`    | 🟢 绿色 | 成功状态    |
| `WARN`  | 🟡 黄色 | 警告信息    |
| `ERR`   | 🔴 红色 | 错误信息    |
| `REQ`   | 🔷 青色 | HTTP 请求 |
| `FILE`  | 🟢 绿色 | 文件传输    |
| `CACHE` | 🟣 紫色 | 缓存命中    |
| `CONF`  | 🟡 黄色 | 配置相关    |

---

## HTTP 状态码颜色

| 颜色    | 状态码  | 含义        |
| ----- | ---- | --------- |
| 🟢 绿色 | 200  | 请求成功      |
| 🟣 紫色 | 304  | 未修改（使用缓存） |
| 🟡 黄色 | 404  | 未找到       |
| 🔴 红色 | 500+ | 服务器错误     |
